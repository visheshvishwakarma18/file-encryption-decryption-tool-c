# File Encryption and Decryption Tool in C

## Description
This project is a simple File Encryption and Decryption Tool developed in C. It allows users to encrypt and decrypt text files using a key-based cipher algorithm.

## Features
- Encrypt text files
- Decrypt encrypted files
- User-defined encryption key
- Simple and easy-to-understand implementation
- Beginner-friendly cyber security project

## How It Works
The program reads data from a text file and applies a key-based character shift to encrypt the contents. The same key is used to decrypt the file and restore the original text.

## Files
- input.txt - Original file
- encrypted.txt - Encrypted output
- decrypted.txt - Decrypted output
- main.c - Source code

## How to Compile

```bash
gcc main.c -o encryptor
```

## How to Run

```bash
./encryptor
```
